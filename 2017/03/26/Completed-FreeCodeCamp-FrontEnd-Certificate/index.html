Completed FreeCodeCamp FrontEnd Certificate

2017-03-26
#extend-script #illustrator

Weekend ended with a front end certification

After about four months, I finally got to and completed the last FrontEnd challenge on Free Code Camp. This was the most fun of all projects and I enjoyed every part of it.
It is an old school Simon Game. With a melodic twist. Every colored area makes a piano sound.
This is the live app: http://liviulvu.github.io/free-code-camp/advanced-frontend-projects/simon-game/

Reaching the end of the front end cours would of happened sooner, but I also have to joggle a full time job and also had a freelance side project on css & html editing.

Things to learn next?

Now I have to find a next growth challenge. And there are a ton of things I want to learn and do next:

  • artificial intelligence
  • big data visualisations
  • backend basics in php and mysql
  • ES6 deeper understanding
  • Code Wars algorithm challenges

That is a bit too much for what my free time allows, so I have to consider priorities.
On my previous freelance projet I had to do some front end changes to css and html.
Since the project was built with Laravel an php, it was a great struggle to populate the tabels I needed to style. This clearly indicates that you can not get away with work on front end without knowing a little bit about back end mechanics.

Until next time, keep learning, keep exploring!

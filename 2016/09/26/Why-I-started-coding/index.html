Why I started coding

2016-09-26
#Coding #Personal

Why i started writing code for the front end?

My interest in coding was sparked by finding that its possible to write code for illustrator and automate things.

I searched for scripts built by others to automate repetitive tasks, but there were cases when i needed to make changes to the code to fit specific needs. I had no idea about coding functionality, so you could say I was shooting in the dark, hoping to get a good result.

In some cases i could not find any scripts related to a task and that pushed me to look into documentation for Illustrator script. It is far from friendly for a beginner with no coding experience.

I dragged along for a feew weeks but the adobe documentation seemed incomplete and tutorials on scripting in Illustrator were very few. So I began to study javascript, completed lots of tutorials and made small projects for the web.

I was enjoying the experience more and more

This also made automation in illustrator a lot easyer to do.
Writing or modifying Illustrator scripts was not needed often, so i began to look more into developing simple websites. Learned basic html, css, javascript, started reading articles about performance optimisations for code maintenance and page load times.

Tried as many tools I could understand and use: text editors, project build automation, frameworks for css and js, github and other stuff related to front end.

The list of things to learn for the front end is a never ending story. Fortunately the developer community is huge and very helpful, starting with Stack Overflow, MDN, codeschool, codeacademy, sites like csstuts and a lot of books and videos.

With such a big jungle I got lost

Learning all neccesary tools to be competitive is a intimidating journey. For finding my way I read articles about what is important to know, learn first and what to set aside for later.

Not sure where this journey will lead me but it gets better each time I learn something new about how to write better code.

What about you? What was it like to learn programming and why you started?
